Getting started with WRIMS2_Scripting


Simple batch example:

(1) Open the file "BatchExmple.bat" with an editor, correct the paths of the config file "CONV_test1.conifg" and "CONV_test2.conifg"

(2) Double-click BatchExample.bat to run the example




Python script example:

(3) If step 2 works, then open the file "MS_template.py" with and editor, correct the paths of CONV.config and TXFR.config on line 11 and 14

(4) Double-click the shortcut "WRIMS2_Scripting" to bring up the command window

(5) Type "jython MS_template.py" in the command window to run the example
